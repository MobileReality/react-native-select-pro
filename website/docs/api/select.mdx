The `<Select>` component has only one required prop, which is the `options` prop.

## Basic example

```SnackPlayer name=Basic&dependencies=@mobile-reality/react-native-select-pro
  import { View } from 'react-native';
  import { Select, SelectProvider } from '@mobile-reality/react-native-select-pro';

  const DATA = [
      {
          label: 'Option 1',
          value: 'option1',
      },
      {
          label: 'Option 2',
          value: 'option2',
      },
      {
          label: 'Option 3',
          value: 'option3',
      },
      {
          label: 'Option 4',
          value: 'option4',
      },
  ];

  const App = () => {
    return (
      <SelectProvider>
        <View style={{ flex: 1, alignItems: 'center', justifyContent: 'center' }}>
          <Select options={DATA} styles={{ width: 300 }} />
        </View>
      </SelectProvider>
    );
  };

  export default App;
```
