The `Select` component has only one required prop, which is the `options` prop.

## Props

### Basic

<div className='table-size'>

| Name                       | Type                          | Default                             | Description                                                                                                                                                                                                                                                | Example                                              |
| -------------------------- | ----------------------------- | ----------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------- |
| `options` **(required)**   | `OptionsType<T>`              |                                     | An array of objects that represents the available options for a `Select`.                                                                                                                                                                                  | <TableExample expo={<OptionsPropSnackLink />} />     |
| `animation`                | `boolean` \| `number`         | `true` \| `200`                     | An options list open-close animation can be enabled or disabled using a `boolean` parameter. The duration of the animation can be set using a `number` parameter. If you pass `0` as the duration, the animation is still occur but will be instantaneous. | <TableExample expo={<Animation />} />                |
| `clearable`                | `boolean`                     | `true`                              | Show a clear button to remove selected option.                                                                                                                                                                                                             | <TableExample expo={<Clearable />} />                |
| `closeOptionsListOnSelect` | `boolean`                     | `true`                              | Close the options list after selected option.                                                                                                                                                                                                              | <TableExample expo={<CloseOptionsListOnSelect />} /> |
| `defaultOption`            | `OptionType<T>`               |                                     | An object that represents the default option for a `Select`.                                                                                                                                                                                               | <TableExample expo={<DefaultOption />} />            |
| `disabled`                 | `boolean`                     | `false`                             | Disable a `Select` pressable.                                                                                                                                                                                                                              | <TableExample expo={<Disabled />} />                 |
| `hasBackdrop`              | `boolean`                     | `true`                              | Determines whether a `Select` component should have a `Backdrop` component.                                                                                                                                                                                | <TableExample expo={<HasBackdrop />} />              |
| `hideArrow`                | `boolean`                     | `false`                             | Determines the arrow icon should be displayed.                                                                                                                                                                                                             | <TableExample expo={<HideArrow />} />                |
| `multiple`                 | `boolean`                     | `false`                             | Determines whether a `Select` component should allow the user to select multiple options.                                                                                                                                                                  | <TableExample expo={<Multiple />} />                 |
| `noOptionsText`            | `string`                      | `No options`                        | Determines the text that should be displayed when there are no available options in a `Select` component.                                                                                                                                                  | <TableExample expo={<NoOptionsText />} />            |
| `placeholderText`          | `string`                      | `Select...`                         | Determines the placeholder text that should be displayed in a `Select` component when no option is selected.                                                                                                                                               | <TableExample expo={<PlaceholderText />} />          |
| `placeholderTextColor`     | `string`                      | `#808080`                           | Determines the color of the placeholder text in a `Select` component.                                                                                                                                                                                      | <TableExample expo={<PlaceholderTextColor />} />     |
| `scrollToSelectedOption`   | `boolean`                     | `true`                              | Determines whether a `Select` component should automatically scroll to the selected option when the options list is opened.                                                                                                                                | <TableExample expo={<ScrollToSelectedOption />} />   |
| `pressableSelectedOption`  | `boolean`                     | `true`                              | Determines whether the selected option should still be pressable after it has been selected.                                                                                                                                                               | <TableExample expo={<PressableSelectedOption />} />  |
| `searchable`               | `boolean`                     | `false`                             | Determines whether a `Select` component should include a search field that allows the user to filter the options by keyword.                                                                                                                               | <TableExample expo={<Searchable />} />               |
| `searchPattern`            | `(payload: string) => string` | `(payload: string) => (${payload})` | Regex definition for searching options.                                                                                                                                                                                                                    | To Do                                                |
| `styles`                   | `SelectStyles`                |                                     | An object that represents the styles for a `Select` component.                                                                                                                                                                                             | To Do                                                |

</div>
