## Requirements

-   `react-native >= 0.63.0`

## Installation

Yarn:
```shell
yarn add @mobile-reality/react-native-select-pro
```

npm:
```shell
npm install @mobile-reality/react-native-select-pro
```

pnpm:
```shell
pnpm add @mobile-reality/react-native-select-pro
```
